Golden Paws & Company — Site

This is a small static site for the Golden Paws & Company dog walking service.

Preview locally:

```bash
cd "golden-paws-site"
python3 -m http.server 8000
# open http://localhost:8000
```

Deploy suggestions:

- GitHub Pages: push this folder to a repo and enable Pages (branch: main or gh-pages).
- Netlify/Vercel: connect the repo and deploy — both detect static sites automatically.

Notes:
- Replace placeholder contact email in `index.html` with your real address.
- Add real images to `hero` and social links in the footer.
